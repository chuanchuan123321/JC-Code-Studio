## 项目介绍

**JC-Code Studio** 是一款前沿的、原生 AI 驱动的前端开发平台，旨在彻底革新开发者构建 Web 应用程序的方式。与 Google AI Studio 类似，JC-Code Studio 提供智能化的对话式编码体验，赋能开发者通过自然语言交互创建令人惊叹的 Web 项目。

JC-Code Studio 诞生于让编程变得简单高效的愿景，利用先进的 AI 模型实时生成、修改和重构前端代码。无论您是经验丰富的开发者还是刚刚踏上编码之旅的新手，JC-Code Studio 都能适应您的工作流程，以前所未有的速度加速开发。

## 核心特性

### 🤖 原生 AI 驱动开发

- **对话式编码界面**：使用自然语言与代码库交互。描述您的需求，看着 JC-Code Studio 将其变为现实
- **实时流式响应**：AI 逐字符流式输出响应，代码即时生成
- **智能上下文感知**：AI 理解您的整个项目结构，确保代码生成的一致性和连贯性
- **多轮对话**：在多次交互中保持上下文，支持复杂项目的开发

### 📁 多文件项目生成

- **自动文件组织**：JC-Code Studio 智能地创建和整理文件到专业的文件夹结构中
- **同时多文件编辑**：单个请求中更新多个文件（HTML、CSS、JavaScript）
- **智能依赖管理**：AI 理解文件依赖关系，按正确顺序加载脚本
- **分层文件夹支持**：创建具有嵌套文件夹和子目录的复杂项目
- **基于模板的生成**：使用专业项目模板或从零开始构建

### 🎨 完整前端技术栈支持

- **HTML 结构生成**：符合语义化标记和可访问性最佳实践
- **CSS 样式设计**：包括动画、响应式设计和高级效果在内的现代样式
- **JavaScript 功能实现**：动态交互、事件处理和复杂逻辑
- **TypeScript 支持**：生成类型安全的代码，构建健壮的应用程序
- **框架无关性**：可与原生 JavaScript 配合使用，或扩展支持框架

### 🔄 高级版本控制

- **代码历史跟踪**：每个 AI 生成的版本都自动保存并带时间戳
- **时光旅行调试**：单击一下即可恢复代码的任何先前版本
- **基于消息的历史**：历史记录与聊天消息关联，便于参考
- **差异可视化**：精确查看版本之间的更改
- **类似分支的开发**：安全地实验，知道始终可以回退

### 📥 代码导入导出

- **拖放导入**：通过将文件拖入工作区轻松导入现有项目
- **项目归档**：将完整项目保存到本地存储以备后用
- **导出功能**：将整个项目下载为 zip 文件
- **剪贴板集成**：快速复制粘贴代码片段
- **格式保留**：导入期间保持代码结构

### 🖼️ 多模态 AI 交互

- **图片上传支持**：上传截图、设计稿或模型以指导 AI 生成
- **视觉上下文理解**：AI 分析图片以生成匹配的 UI 组件
- **设计到代码**：自动将视觉设计转换为功能性代码
- **参考图片对话**：与 AI 讨论上传的图片以精确实现
- **多格式支持**：JPG、PNG、WebP 等

### 🎛️ 专业 IDE 功能

- **语法高亮**：由 Prism.js 提供的美观代码显示
- **分栏布局**：可自定义的工作区，包含聊天、编辑器和预览面板
- **可调整大小面板**：拖动面板边框创建完美工作区
- **基于标签的文件导航**：轻松切换多个打开的文件
- **自动保存**：自动项目持久化，永不丢失工作
- **键盘快捷键**：通过快速操作加快工作流程

### 🚀 实时预览环境

- **即时渲染**：在集成预览面板中立即看到更改
- **隔离执行**：预览在安全的 iframe 环境中运行
- **热重载**：更改立即反映，无需手动刷新
- **响应式测试**：在不同视口大小下测试设计
- **控制台访问**：使用集成浏览器控制台调试 JavaScript
- **性能监控**：跟踪资源使用和加载时间

## 快速入门指南

### 前置要求

- 已安装 Node.js 18+
- 现代化 Web 浏览器（Chrome、Firefox、Safari、Edge）
- 对 Web 开发概念的基本了解

### 安装步骤

1. **克隆仓库**
```bash
git clone https://github.com/your-username/vibe-coding.git
cd vibe-coding
```

2. **安装依赖**
```bash
npm install
```

3. **配置 API 访问**

在项目根目录创建 `.env.local` 文件：

```env
# API 配置
API_URL=https://yunwu.ai/v1
API_KEY=你的API密钥
MODEL=gemini-3-pro-preview

# 可选：自定义 AI 行为
TEMPERATURE=0.7
MAX_TOKENS=32000
```

4. **启动 JC-Code Studio**
```bash
npm run dev
```

5. **开始创作**

在浏览器中打开 `http://localhost:3000`，开启您的 JC-Code Studio 之旅！

## 故障排除

### 常见问题

**问：预览面板显示空白屏幕**
- 确保您有 `index.html` 文件
- 检查浏览器控制台是否有 JavaScript 错误
- 尝试刷新预览

**问：AI 响应缓慢**
- 检查您的互联网连接
- 验证 `.env.local` 中的 `API_KEY` 是否正确
- 尝试在配置中减少 `MAX_TOKENS`

**问：无法上传图片**
- 确保图片格式受支持（JPG、PNG、WebP）
- 检查文件大小（建议最大 10MB）
- 验证浏览器权限

**问：丢失代码更改**
- 检查历史面板并恢复先前版本
- 在 localStorage 中查找自动保存的项目
- 检查浏览器缓存以获取最近的更改
